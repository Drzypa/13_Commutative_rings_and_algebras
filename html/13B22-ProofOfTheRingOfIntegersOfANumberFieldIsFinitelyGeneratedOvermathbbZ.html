proof of the ring of integers of a number field is finitely generated over ℤ


Proof:  Choose any basis α1,…,αn of K over ℚ. Using the theorem in the entry multiples of an algebraic number, we can multiply each element of the basis by an integer to get a new basis α1,…,αn with each αi∈𝒪K.
Consider the group homomorphism

φ:K→ℚn:γ↦(TrℚK⁡(γ⁢α1),…,TrℚK⁡(γ⁢αn))

where TrℚK is the trace (http://planetmath.org/trace2) from K to ℚ. Note that φ is 1-1, since if γ≠0 and φ⁢(γ)=0, then

n=TrℚK⁡(1)=TrℚK⁡(γ⁢γ-1)=TrℚK⁡(γ⁢∑ri⁢αi)=∑ri⁢TrℚK⁡(γ⁢αi)=0

where the last equality holds since γ∈ker⁡φ.

Hence φ:𝒪K↪ℤn, so 𝒪K is finitely generated and torsion-free. It has rank ≥n since the αi are linearly independent, and rank ≤n since it injects into ℤn, so it has rank n.
