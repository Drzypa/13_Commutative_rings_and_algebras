non-isomorphic completions of ℚ


No field ℚp of the p-adic numbers (p-adic rationals (http://planetmath.org/PAdicIntegers)) is isomorphic with the field ℝ of the real numbers.

Proof.  Let’s assume the existence of a field isomorphism  f:ℝ→ℚp  for some positive prime number p.  If we denote  f⁢(p)=a,  then we obtain

a2=(f⁢(p))2=f⁢((p)2)=f⁢(p)=p,

because the isomorphism maps the elements of the prime subfield on themselves.  Thus, if  |⋅|p  is the normed p-adic valuation (http://planetmath.org/PAdicValuation) of ℚ and of ℚp, we get

|a|p=|a2|p=|p|p=1p,

which value is an irrational number as a square root of a non-square (http://planetmath.org/SquareRootOf2IsIrrationalProof) rational.  But this is impossible, since the value group of the completion ℚp must be the same as the value group |ℚ∖{0}|p which consists of all integer powers of p.  So we conclude that there can not exist such an isomorphism.
